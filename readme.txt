2.4G无线通信：
使用定时器计数值作为随机种子产生20个信道跳频通信,每次发送跳一次,循环往复
使用CPU唯一ID生成4字节通信地址,接收器收到并记录该地址实现配对
接收跳频图案以及配对时使用固定信道2520MHz
配对时使用地址0x59414E47
空中波特率250Kbps
自动重发最大5次
2字节CRC
10个字节数据发送：1byte左摇杆Y自动回中标志+1byte8个按键+2byte左摇杆X(-200~200)+2byte左摇杆Y(-200~200/ 0~400)+2byte右摇杆X(-200~200)+2byte右摇杆Y(-200~200)
4个字节数据发送(配对时)：使用CPU唯一ID生成4字节通信地址
20字节数据发送(发送跳频图案时)：跳频图案
无线通信周期22ms

开关机：
红色圆形按钮长按1s蜂鸣一声开机、长按1s蜂鸣一声关机

配对模式：
长按开机后保持开机按钮按下持续2s蜂鸣一声进入配对模式
成功发送地址(配对成功)后蜂鸣一声进入发送跳频图案模式
此模式红蓝LED同时流水灯

发送跳频图案模式：
正常开机默认进入发送跳频图案模式
发送跳频图案成功后进入发送模式
此模式蓝色LED流水灯

发送模式：
发送连续失败超时(1S)进入发送跳频图案模式
此模式发送成功蓝色LED，失败红色LED

电量指示：
开关机按钮周围LED指示电量

自动关机：
发送模式以及发送跳频图案模式摇杆无任何动作或者发送跳频失败持续5min则自动关机
配对模式持续5min则自动关机
关机前红灯闪烁30s
